▶️ How It Works
ESP32 reads tilt data from MPU6050
Converts motion into movement
Renders game on OLED display
You control everything by tilting 😄

🧪 Troubleshooting

Screen not working?

Check I2C address (0x3C or 0x3D)

Gyroscope not responding?

Verify wiring (SDA/SCL)
Try another MPU6050 library

Game lagging?

Reduce OLED refresh rate
